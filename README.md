# VRKit
Swift IOS Pod. VRKit turns your app into Virtual Relity App.

Import:

        import VRKit

Cocoapods:

        pod 'VRKit'

1) To use it simply call start func in your UIViewController

        VR.start()
        
1) To stop VR use 

        VR.stop()
